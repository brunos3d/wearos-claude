# Code of Conduct

This project follows the spirit of the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Short version

- Be respectful. Disagreeing is fine; being a jerk is not.
- Assume good faith. Most "obviously wrong" suggestions in open source are
  someone working with a different context than you have.
- No harassment, discrimination, doxxing, or personal attacks. Anywhere
  (issues, PRs, discussions, commit messages).
- This is a hobby project. Be patient with response times — and with
  contributors learning Wear OS, Kotlin, or TypeScript in public.

## Reporting

If someone violates this, email **bruno3dcontato@gmail.com** with a brief
description and any relevant links. Reports stay private to the maintainer.

Maintainers may remove comments, close issues/PRs, or block contributors who
violate this code of conduct.
